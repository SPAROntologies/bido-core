Introduction

The core module of BiDO allows us to describe any entity and the related bibliometric data (through the property holdsBibliometricDataInTime) at a certain time (i.e., tvc:atTime, a property defined by the imported TVC ontology for specifying temporal instants or intervals) and according to a certain agent (through the property accordingTo, which is a sub-property of prov:wasAttributedTo and allows us to indicate the agent responsible for such bibliometric data). Two alternative kinds of bibliometric data are specifiable (through the property withBibliometricData) in BiDO: numeric data and categorial data. Numeric data are those characterised by a certain integer or float value related to a particular bibliometric measure. Some of these measures â i.e., h-index, author citation count, e-index, and journal impact factor â are available in a particular module of BiDO responsible for describing the most common bibliometric measures.
